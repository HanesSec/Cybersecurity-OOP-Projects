Meus Labs de Python & Segurança
Scripts em Python que desenvolvi para aplicar Programação Orientada a Objetos (POO) em cenários de cibersegurança e monitoramento de infraestrutura. Esse repositório serve para organizar meus estudos da faculdade e meus laboratórios pessoais.

 O que você vai encontrar aqui:
Defesa de Rede: Simuladores de Firewall, filtros de pacotes e detecção de intrusos.

Controle de Acesso: Lógica para cofres digitais e sistemas de portaria com níveis de permissão.

Monitoramento: Automação para checar temperatura e umidade em racks e inventário de hardware.

Tecnologias e Estudos:
Atualmente estou cursando Segurança da Informação e focando muito em Python, Linux (Ubuntu) e nas trilhas da Cisco NetAcad. Uso esses scripts para fixar os conceitos de como a programação pode automatizar a defesa de ambientes críticos.
