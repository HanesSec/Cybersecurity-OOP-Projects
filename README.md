# Cybersecurity OOP Projects

Estudos de **Programação Orientada a Objetos** com temática de segurança da informação.  
Cada arquivo aplica conceitos de POO — herança, encapsulamento e polimorfismo — em cenários reais de infraestrutura e defesa de redes.

<br>

## // arquivos

| Arquivo | Conceito POO | Cenário |
|--------|--------------|---------|
| `access_control_system.py` | Herança | Controle de acesso por cargo e setor |
| `security_device_controller.py` | Encapsulamento | Cofre digital com bloqueio por tentativas |
| `hardware_inventory_manager.py` | Herança + polimorfismo | Inventário de periféricos de TI |
| `infrastructure_monitor.py` | Polimorfismo | Sensores de temperatura e umidade em racks |
| `firewall_ip_manager.py` | Herança | Firewall com lista de IPs bloqueados |
| `network_defense_system.py` | Herança múltipla | IDS + filtro de pacotes combinados |

<br>

## // conceitos aplicados

`herança` `herança múltipla` `encapsulamento` `polimorfismo` `classes` `métodos`

<br>
